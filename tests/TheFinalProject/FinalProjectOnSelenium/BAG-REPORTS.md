# Баг-репорты:
## 1. Проверка регистрации в бонусной системе и повторного использования промокода
### Шаги:
1. Перейти в техспецификацию по проекту 
2. Спустить вниз техспецификации и изучить флоу с бонусной системой 

### Ожидаемый результат:
* Описание сценария содержит подробную и понятную информацию
### Фактический результат: 
* Описание сценария неполное и нуждается в уточнении

## 2. Отсутствие текста сообщений об ошибках в DOM-модели при валидации полей
### Шаги:
1. Перейти на страницу бонусной программы
2. Не заполнять обязательные поля "Имя" и "Телефон"
3. Нажать кнопку "Оформить карту"
4. Проверить отображение ошибок в консоли или черзе локаторы автотестов

### Ожидаемый результат:
* Текст ошибки присутствует в DOM и доступен для логирования
### Фактический результат: 
* Текст визуально отображается на UI, но возвращает пустое значение при обращении через локаторы, что указывает на проблему с доступностью элементов для автотестов
